Laam

Laam is a native token built on Stellar, designed to be a modern solution for digital currency. It has a fixed supply minted only once to preserve scarcity and long-term value.

Laam Wallet Explorer

Laam is an open project aimed at supporting real-world use cases such as property sales, vehicle rentals, and community-based commerce. It can be easily added to popular wallets by creating a trustline, including Lobstr, Solar, and Freighter.

The token is designed to be secure, non-inflationary, and fully tradable.
